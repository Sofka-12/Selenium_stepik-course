# Selenium_stepik-course
Дополнение по результату выполнения курса  «Автоматизация тестирования с помощью Selenium и Python» в рамках учебной программы.

Курс доступен к прохождению по ссылке https://stepik.org/575

Репозиторий содержит коды выполненных заданий по тестированию с использованием Selenium. Файлы именуются в соответствии с номерами заданий курса, требующих написания кода.
